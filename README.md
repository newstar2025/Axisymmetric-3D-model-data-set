# Axisymmetric-3D-model-data-set

**1. Dataset Overview**

This dataset is constructed based on basic models from public resources through manual modeling and parametric modification. It focuses on typical axisymmetric 3D models in the field of mechanical manufacturing, covering core fastener and assembly parts, and can be used for academic research such as model retrieval, recognition, and optimization in related fields.
Based on four typical axisymmetric basic models, this dataset is modified and expanded, namely Gear, Wheel, Nut, and Knob, all of which are representative part types in the field of mechanical manufacturing.

Total scale of the dataset: 1,650 models, divided into 15 subcategories, with each subcategory containing 110 models evenly. The data distribution is uniform, which can meet the needs of diverse academic research.
